# Data Access Instructions

## OHTS Data

Available through NIH dbGaP: accession `phs000240.v1.p1`

### Steps
1. Visit https://www.ncbi.nlm.nih.gov/gap/
2. Apply for authorized access
3. Place downloaded files in `data/raw/`

### Reference
Gordon MO et al. (2002) *Archives of Ophthalmology*, 120(6):714-720.
